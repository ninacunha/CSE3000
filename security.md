# Security Considerations

## Intended Users
The intended users of this repository are the course instructor, teaching assistants, and the student who owns the repository. The repository contains coursework and supporting code created for the class.

## Assessment of Security Risks
The security risks associated with this repository are relatively low because it only contains coursework and example code. However, possible risks include unauthorized modification of code or misuse of the code if it were copied and used elsewhere. No sensitive data, passwords, or API keys are stored in this repository.

## Steps Taken to Secure the Repository
Several steps have been taken to protect the repository:

- A GitHub ruleset is applied to the default branch to manage pull requests.
- Pull request rules are enabled to ensure that changes go through the pull request process.
- Code owner reviews are required before certain changes can be merged.
- Branch protections prevent direct deletion or non-fast-forward changes to the default branch.
- No sensitive credentials or personal data are stored in the repository.
